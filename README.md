## 浮生

### 简介

浮生是一款记录生活与爱情琐事的软件。

在这里你可以记录你看过哪些电影、网络小说、图书与玩过哪些游戏；

在这里你可以记录你与伴侣的点点滴滴，比如一起在电影院看过的电影、纪念日、旅行地、生活琐事；

除这些以为，还会陆续添加模块。
